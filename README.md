# Blog Universitário de Tecnologia

## Projeto pessoal de um sistema de blog onde os usuários podem postar conteúdos e comentários sobre tecnologia

### Tecnologias 
* Frontend
  * Python
  * Django
  * Tailwind
  * HTML
  * CSS
* Backend
  * PHP
  * PHPixie
* Banco de Dados e Integrações
  * ORM
  * PostgreSQL
  * APIs Rest
